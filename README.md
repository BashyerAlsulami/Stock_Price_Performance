# Stock_Price_Performance

## Topic chosen 
The topic of this dashboard is Stock Price Performance.
It is designed to help stakeholders analyze daily price movements, returns, and fluctuations over time, providing insights for better investment decisions.

## The data
Data was obtained from Kaggle: https://www.kaggle.com/datasets/ibrahimshahrukh/mercedes-benz-stock-prices-1996-2026

The dataset contains daily stock market data for Mercedes-Benz Group, including Open, High, Low, Close prices, daily returns, and volume.

## Data cleaning
Data preparation steps:

Dropped unnecessary rows
Checked and handled null values
Filled missing values 

All steps are documented in Data_Cleaning.ipynb

## Main takeaways: 

- Stock prices show long-term upward trends with periodic volatility spikes.  
- Daily returns fluctuate significantly some days have large negative or positive movements.  
- Stakeholders can use this dashboard to:
  - Identify periods of high volatility  
  - Monitor daily and yearly performance  
  - Make informed investment decisions based on trends
